# 佐久間 龍星 – 君を誰よりも

> **歌詞登録ステータス：プレーン歌詞のみ**

```
どうして胸の痛くなる場所で人は恋をしてる?
君を想えば想う度に空が滲んでく
気付いてた 君が苦しくて泣いてた事を
それなのに僕は何ひとつも向き合えず
夢ばかり追いかけた
溢れる程 君は愛を注ぎ続けていたのに
君が愛した僕は君を誰よりも傷付けた人
"好きだよ"なんて素直に言えず それが僕の弱さ
"好き"なんて言葉なくても側に居た 君の強さ
溢れる程 君は愛を注ぎ続けていたのに
君が愛した僕は君を誰よりも傷付けた人
その傷が僕じゃない恋で癒す事ができたなら
君の幸せが僕の中で答えになり始める
溢れる程 君は愛を注ぎ続けていたから
これからは愛で溢れる誰よりも愛される人
```
